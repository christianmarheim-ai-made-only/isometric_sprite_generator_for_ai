# red_dragon_v1

Generated-data package derived from the approved creature concept art.

Included:
- starter GLB
- source-side descriptor
- model source spec
- animation rawdata
- hitbox data
- physical metrics
- spell orbit metadata
- materials and sockets
- concept art reference
- generator script stub

Contract note: current external_asset_v1 only supports `biped` and `bird`, so this package does not include a schema-valid external asset manifest. A custom `dragon_v1` rig profile is shipped instead.
