# pirate_duelist_v2

Complete generated-data package for the pirate duelist character.

Included:
- starter GLB
- external_asset_v1 manifest
- source asset descriptor
- model source spec
- animation rawdata
- hitbox data
- physical mass/weight metrics
- sockets
- pelvis and shoulder spell-orbit ring metadata
- combat hooks for muzzle, reload, footsteps, and attachment points
- materials and texture atlas
- concept art references
- generator script
- validation report

The manifest and rawdata use current playback vocabulary: `loop` and `once`.
